## 🧑‍💻 Sobre mí

> "*El código es arte, la personalización es libertad.*" - *KDE/Mac Inspired philosophy*

Soy un apasionado estudiante de **Desarrollo de Aplicaciones Web (DAW)** en **CEI Sevilla**. Busco la excelencia técnica con un ojo puesto en la estética y la experiencia de usuario. Mi entorno de trabajo actual es **Fedora 43 con KDE Plasma 6**, personalizado para alcanzar la armonía perfecta entre funcionalidad y diseño visual.
